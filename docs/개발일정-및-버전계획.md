## 📆 개발 일정 및 버전 계획

### 📌 개발 목표 요약
- **프로젝트 명**: TR-Manager
- **목표**: Revit 모델 내 Room 요소 기반 주택 정보 관리 프로그램 (세대/층/건물 단위 관리)
- **방식**: Revit Add-In + WPF 기반 독립 GUI
- **목표 마감**: 2025년 7월 말 ~ 8월 초까지 인수 테스트 완료 상태 도달
